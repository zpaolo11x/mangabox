Features Highlights

Rich Experience Read in two spreads with book effects and page turn animation, supports webtoons and reading directions.
Page Thumbnails Reader interface features page thumbnails and seek controls.
Sync Progress Sync reading progress with your server on all devices, even when reading offline.
Global Search Search your series and books all in one place; looks into metadata, summary etc.
Offline Reading Download your books and read theme wherever you are, even with no connection to the server.
Custom Themes Adapt to your taste with dark and light theme, plus custom color highlights.
Live Sort & Filter Sort your library on the fly or filter by read/unread, reading direction, language etc.
Advanced Tagging Tag your books in Komga: "no-cover" to start with a page spread, "no-blank-page" to add a fake back of the cover.
Advanced Metadata Add alternate titles in Komga: "Story" and "Art" to set a series authors, "Time" for series dates
Advanced Year Search A special search prefix "y:" will show all the series that have volumes in that date range or year.

Installation notes

  • To deploy the web verions just download the "web" release artifact and unpack it in a "mangabox" folder on your web server. Because of CORS limitation by default MangaBox will work only if it's on the same location as your komga server.
  • Electron version is not limited by CORS and it will work with any Komga server.
  • The macOS binaries are not signed, so they won't run by default. You can allow them to run on Intel macs by right-click/open but on Apple Silicon you'll need to ad-hoc sign the binary. This is a simple bash script that will do the hard work for you:
    #!/bin/bash
     app="$1"
     xattr -d com.apple.quarantine "${app}"
     find "${app}/Contents" -type f -exec codesign --force --timestamp --verify --verbose --sign - "{}" \;
     codesign --force --timestamp --verify --verbose --sign - "${app}"
    Just chmod +x the script and run it with the .app file path as argument.
  • Mobile versions are not signed, you can sideload the Android apk on your device, and you can sideload the iOS/ipadOS version using a developer tool like AltStore which will self sign and renew the signature every 7 days.
