Features Highlights

Rich Experience Read in two spreads with book effects and page turn animation, supports webtoons and reading directions.
Page Thumbnails Reader interface features page thumbnails and seek controls.
Sync Progress Sync reading progress with your server on all devices, even when reading offline.
Global Search Search your series and books all in one place; looks into metadata, summary etc.
Offline Reading Download your books and read theme wherever you are, even with no connection to the server.
Custom Themes Adapt to your taste with dark and light theme, plus custom color highlights.
Live Sort & Filter Sort your library on the fly or filter by read/unread, reading direction, language etc.
Advanced Tagging Tag your books on the server: "no-cover" to start with a page spread, "no-blank-page" to add a fake back of the cover.
Advanced Metadata Add alternate titles on the server: "Story" and "Art" to set a series authors, "Time" for series dates
Advanced Year Search A special search prefix "y:" will show all the series that have volumes in that date range or year.

Localization

MangaBox is available in different languages and uses Weblate to manage translation files. Feel free to contribute to the project by submitting corrections and translations at https://hosted.weblate.org/projects/mangabox/.

Installation notes

  • To deploy the web verions just download the "web" release artifact and unpack it in a "mangabox" folder on your web server. Because of CORS limitation by default MangaBox will work only if it's on the same location as your server.
  • Electron version is not limited by CORS and it will work with any server location.
  • Android version is not signed, you can sideload the apk on your device.
